File Download

        URL-Format:  /file/{generator => random | zero}/{filesize}/{unit => b | kb | mb | gb | tb; default: mb}
        URL-Example: {{ url_for('generate_file', generator='random', sz=1, unit='mb') }}
        Optional HTTP-GET Parameters:
          - chunk_size = bytes to read from content generator respectively to send per iteration (default: 1000)
          - timeout = seconds until connection is closed on server side (default: two minutes per gigabyte. max: 1h)

Usage Examples

            Using curl:     curl --output /dev/null {{ url_for('generate_file', generator='random', sz=1, unit='gb', _external=True) }}
            Using wget:     wget -O /dev/null [--report-speed=bits] {{ url_for('generate_file', generator='random', sz=1, unit='gb', _external=True) }}
