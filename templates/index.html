[=====filespeed===============>          ]

File Download

        URL-Format:  /file/{generator => random | zero}/{filesize}/{unit => b | kb | kib | mb | mib | gb | gib | tb | tib; default: mb}
        URL-Example: {{ url_for('generate_file', generator='random', sz=1, unit='mb') }}
        Optional HTTP-GET Parameters:
          - chunk_size = bytes to read from content generator respectively to send per iteration (default: 8192)
          - timeout = seconds until connection is closed on server side (default: six minutes per gigabyte. max: 1h)

Generator: Random

  • 500 KB
  • 1 MB
  • 500 MB
  • 1 GB
  • 10 GB

Generator: Zero

  • 500 KB
  • 1 MB
  • 500 MB
  • 1 GB
  • 10 GB

Usage Examples

            Using curl:     curl --output /dev/null {{ url_for('generate_file', generator='random', sz=1, unit='gb', _external=True) }}
                            # Optional parameters:
                            #   --http1.1     -> force http version 1.1 (mostly default)
                            #   --http2       -> force http version 2 which supports multiplexing

            Using wget:     wget -O /dev/null {{ url_for('generate_file', generator='random', sz=1, unit='gb', _external=True) }}
                            # Optional parameters:
                            #   --report-speed=bits     -> report measured rate in bit rather than bytes per second
